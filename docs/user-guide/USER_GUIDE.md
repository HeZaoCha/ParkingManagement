## 系统概述

早点喝茶停车场管理系统是一套完整的停车场运营解决方案，支持：

- 🚗 车辆入场/出场管理
- 💰 停车费用自动计算
- 📊 实时数据监控
- 🎫 VIP/员工免费停车
- 👥 多角色权限管理

### 系统入口

| 入口 | URL | 说明 |
|------|-----|------|
| 工作人员登录 | `/login/` | 日常操作界面 |
| 管理后台 | `/manage/` | 数据管理 |
| 客户自助 | `/parking/customer/` | 车辆查询 |

## 工作人员操作

### 登录系统

1. 访问登录页面 `/login/`
2. 输入用户名和密码
3. 点击"登录"按钮

![登录页面](images/login.png)

### 工作台界面

登录后进入工作台，显示：
- 今日统计（入场数、出场数、收入）
- 停车场状态（空闲车位/总车位）
- 最近停车记录

### 车辆入场

**方式一：快捷入场**

1. 点击顶部导航栏"快捷入场"按钮
2. 输入车牌号（如：粤E9KM03）
3. 选择停车场
4. 选择车辆类型（可选）
5. 点击"确认入场"

**方式二：通过管理后台**

1. 进入"管理后台" > "停车记录"
2. 点击"新建记录"
3. 填写入场信息

### 车辆出场

**方式一：快捷出场**

1. 点击顶部导航栏"快捷出场"按钮
2. 输入车牌号
3. 系统显示停车时长和费用
4. 确认收费后点击"确认出场"

**方式二：扫描查询**

1. 点击"车辆查询"
2. 输入车牌号
3. 查看停车详情
4. 点击"办理出场"

### 车牌号输入规范

系统支持中国大陆车牌号：

| 类型 | 格式 | 示例 |
|------|------|------|
| 普通车牌 | 省份+地市+5位 | 粤E9KM03 |
| 新能源 | 省份+地市+D/F+5位 | 粤ED12345 |
| 挂车 | 省份+地市+4位+挂 | 粤E1234挂 |

⚠️ **注意**：输入时自动转换为大写，忽略空格

### VIP/员工车辆

系统自动识别VIP和员工车辆：
- 入场时显示"VIP车辆"标识
- 出场时自动免除停车费
- 工作台显示特权类型

## 管理员操作

### 进入管理后台

1. 登录系统
2. 点击顶部"管理后台"链接
3. 或直接访问 `/manage/`

### 停车场管理

**查看停车场列表**
1. 点击左侧菜单"停车场管理"
2. 查看所有停车场信息

**添加停车场**
1. 点击"新建停车场"
2. 填写信息：
   - 名称（如：早点喝茶停车场）
   - 地址
   - 总车位数
   - 每小时费率
3. 点击"保存"

**编辑停车场**
1. 点击停车场行的"编辑"按钮
2. 修改信息
3. 点击"保存"

### 车位管理

**批量创建车位**
1. 进入停车场详情
2. 点击"批量创建车位"
3. 输入起始编号和数量
4. 选择车位类型
5. 点击"创建"

**车位类型**
| 类型 | 说明 |
|------|------|
| 标准 | 普通车位 |
| VIP | 贵宾专用 |
| 大型 | 货车/大型车辆 |
| 残疾人 | 无障碍车位 |

### 车辆管理

**查看车辆列表**
1. 点击"车辆管理"
2. 支持按车牌号搜索

**编辑车辆信息**
1. 点击车辆行的"编辑"
2. 可修改车主姓名、电话等

### VIP车辆管理

**添加VIP车辆**
1. 点击"VIP车辆管理"
2. 点击"添加VIP车辆"
3. 输入车牌号
4. 选择特权类型（员工/VIP客户/合作伙伴）
5. 填写车主信息
6. 点击"保存"

**停用VIP特权**
1. 找到VIP车辆记录
2. 点击"编辑"
3. 取消"启用"状态
4. 保存

### 停车记录管理

**查看停车记录**
1. 点击"停车记录"
2. 支持筛选：
   - 按车牌号
   - 按状态（在场/已出场）
   - 按日期范围

**手动结算**
1. 找到未结算的记录
2. 点击"详情"
3. 确认费用后点击"完成结算"

## 客户自助服务

### 访问客户端

访问 `/parking/customer/` 进入客户自助服务页面

### 查询停车状态

1. 在搜索框输入车牌号
2. 点击"查询"
3. 显示结果：
   - 在场：显示入场时间、车位号、当前费用
   - 不在场：显示上次停车信息

### 查看停车场状态

客户端首页显示：
- 停车场名称
- 空闲车位数
- 每小时费率

## 费用说明

### 计费规则

1. **免费时段**：入场后15分钟内免费
2. **计费单位**：按小时计费，不足1小时按1小时算
3. **费率**：由各停车场设定（默认5元/小时）

### 计费示例

| 停车时长 | 费用（5元/小时） |
|---------|----------------|
| 10分钟 | 0元（免费） |
| 30分钟 | 5元 |
| 1小时 | 5元 |
| 1小时30分 | 10元 |
| 3小时 | 15元 |

### VIP/员工免费

- VIP车辆入场自动识别
- 出场时费用显示为0元
- 记录中标记"免费停车"

## 常见问题

### Q: 忘记密码怎么办？

联系系统管理员重置密码。

### Q: 车牌号输入错误怎么办？

联系管理员在"停车记录"中修改车牌号信息。

### Q: 系统显示"车辆已在场"但实际已离开？

1. 进入管理后台
2. 找到该车辆的停车记录
3. 手动办理出场

### Q: 如何添加新的工作人员账号？

1. 管理员登录
2. 进入管理后台
3. 点击"用户管理"
4. 添加新用户并分配"工作人员"角色

### Q: 停车场显示已满但实际有空位？

1. 检查是否有"僵尸车辆"（长期未出场）
2. 进入"车位管理"检查车位状态
3. 手动更新错误的车位占用状态

### Q: 如何导出停车记录？

1. 进入"停车记录"
2. 设置筛选条件
3. 点击"导出"按钮
4. 选择导出格式（Excel/CSV）
